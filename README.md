# manjaro-juno-tablet
Manjaro drivers for Juno Tablet

## Compile Manually

```sudo pacman -S make```

```makepkg -si```

## Install from Manjaro's Repo

```sudo pacman -S juno-tablet```

## Kernel 6.1
Requires ```kernel 6.1``` otherwise sound will not work after ```Hibernate```.

```sudo pacman -S linux61```
